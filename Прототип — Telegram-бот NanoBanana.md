## 1. ЧТО ДЕЛАЕТ БОТ

Бот генерирует профессиональные фотографии женского белья (трусы) через NanaBanana Pro на платформе Replicate. Анализирует референсы и текстовые описания через Claude Sonnet, составляет промпты и запускает генерацию. Без фото трусов генерация не запускается — жёсткое условие.

**Бренд дополнительных элементов:** INTIMNO

---

## 2. ФУНКЦИИ БОТА

### Функция 1 — «По референсу» (`/reference`)
Пользователь присылает любое понравившееся изображение → Claude анализирует стиль, угол, свет, фон, настроение → составляет **2 варианта промпта** → просит прислать фото трусов → предлагает добавить доп. элементы → запускает **2 генерации параллельно** → возвращает **2 PNG-файла**.

### Функция 2 — «По стилю» (`/style`)
Пользователь присылает фото трусов → выбирает стиль из меню **4 стилей** → Claude генерирует **3 уникальных промпта** по шаблону стиля → предлагаются доп. элементы → бот прогоняет 3 промпта параллельно через NanaBanana Pro → отправляет **3 PNG-файла**.

Промпты генерируются заново каждый раз — поверхности, реквизит и углы каждый раз разные, но всегда в рамках выбранного стиля.

### Функция 3 — «Описать стиль» (`/describe`)
Пользователь описывает стиль текстом (на русском или английском) → Claude составляет промпт → просит прислать фото трусов → предлагает доп. элементы → генерирует **1 PNG-файл**.

### Доп. элементы (во всех функциях)
После получения фото трусов в любой функции бот предлагает добавить в кадр:
- 📇 Визитку с брендом INTIMNO
- 📖 Журнал INTIMNO
- ✏️ Любой реквизит словами
- 🖼 Реквизит фотографией (Claude анализирует и описывает его)
- ✅ Без добавок — сразу к генерации

## 3. ЛОГИКА И СЦЕНАРИИ (FLOWS)

### Flow 1 — По референсу

```
Пользователь → /reference
     ↓
Бот просит прислать референс-изображение
     ↓
Пользователь → присылает референс (фото или файл PNG/JPG/JPEG)
     ↓
Claude анализирует: тип кадра, угол, свет, фон, пропсы, настроение
     ↓
Бот составляет 2 варианта промпта и показывает их пользователю
     ↓
Бот просит прислать фото трусов
     ↓
Пользователь → присылает фото трусов (фото или файл)
     ↓
Бот показывает меню доп. элементов
     ↓
Пользователь → выбирает элемент или «Без добавок»
     ↓
Бот запускает 2 генерации параллельно в NanaBanana Pro
     ↓
Бот отправляет 2 PNG-файла пользователю
```

## 6. СТИЛИ И ПРОМПТЫ

Все промпты: формат 3:4 вертикальный, без людей, только product/flatlay, на выходе PNG 1K.

**Важно:** начиная с апреля 2026 промпты для стилей **генерируются динамически через Claude** при каждом запросе. В `prompts.py` хранятся шаблоны-описания стилей (`STYLE_BLUEPRINTS`) — не готовые промпты, а характеристики: поверхности, реквизит, освещение, углы, настроение. Claude каждый раз придумывает 3 уникальных промпта на их основе.

---

### 🌸 НЕЖНЫЙ — шаблон стиля

Романтичная, женственная boudoir-эстетика.
- **Поверхности:** светлый молочный сатин, белое постельное бельё, лён, кремовая ткань
- **Реквизит:** свежие розы и пионы, сухоцветы, ювелирная подставка, жемчужное ожерелье, букетик
- **Свет:** мягкий рассеянный оконный свет, утренний, воздушный
- **Углы:** 90° флетлэй или 70° полуфлетлэй
- **Настроение:** романтика, Pinterest boudoir, мечтательно, нежно и женственно

---

### 🌑 ТЁМНЫЙ — шаблон стиля

Мрачная, тёмная editorial-эстетика.
- **Поверхности:** тёмно-серый плюшевый мех, нюдовая ткань на тёмном фоне, матовый чёрный поднос, тёмный бархат
- **Реквизит:** свечи с тёплым пламенем, флакон люкс-парфюма, чёрная свеча, тёмный декоративный камень
- **Свет:** драматичный софтбокс под 45°, тёплый вечерний свет, мягкие драматичные тени
- **Углы:** 90° флетлэй или 70° полуфлетлэй
- **Настроение:** мрачный тёмный editorial, высокий Instagram-люкс, чувственно и загадочно

---

### 💎 RICH — шаблон стиля

Роскошная luxury editorial-эстетика.
- **Поверхности:** белое шёлковое кимоно, белый мрамор, слоновая кость бархат, шампанский атлас
- **Реквизит:** золотая цепочка, жемчужные серьги, браслет, люкс-парфюм с золотой крышкой, белые орхидеи, журнал INTIMNO, кристальная шпилька, россыпь жемчуга, книга с золотым тиснением
- **Свет:** яркий золотой солнечный, тёплый студийный с золотыми рефлекторами
- **Углы:** 75–80° полуфлетлэй или 70° lifestyle
- **Настроение:** luxury boudoir, тёплые тона слоновой кости и золота, high-end fashion editorial

---

### 🎲 СМЕШАННОЕ — шаблон стиля

Эклектичная lifestyle-эстетика с неожиданными сочетаниями.
- **Поверхности:** белый атлас, серый современный диван, серый пушистый мех на белой постели, дерево, пастельный лён
- **Реквизит:** нотная тетрадь, крем для тела, букет ромашек, золотая цепочка, свеча в чёрной банке, белая книга, суккулент, старинная чашка, полароид, связка лаванды
- **Свет:** яркий естественный дневной, полоска утреннего солнца, тёплый уютный вечерний
- **Углы:** вид строго сверху, 70° полуфлетлэй или 60° lifestyle — варьируется
- **Настроение:** Pinterest lifestyle, уютно и неожиданно, обаятельная Instagram-эстетика

|-----------|-----------|

## 8. СТРУКТУРА ПРОЕКТА

```
bot-for-images/
├── bot.py          # Хэндлеры, FSM, запуск
├── config.py       # Переменные окружения
├── states.py       # Состояния FSM
├── prompts.py      # 4 шаблона стилей (STYLE_BLUEPRINTS) для динамической генерации промптов
├── services.py     # Вызовы Replicate API
├── .env            # Секреты (не коммитить)
├── .env.example    # Шаблон переменных
├── requirements.txt
└── README.md
```

### Машина состояний (State Machine)

```
ReferenceFlow:
  waiting_reference   → ждёт референс-фото
  waiting_panties     → ждёт фото трусов
  choosing_extras     → показано меню доп. элементов
  waiting_extras_text → ждёт текстовое описание реквизита
  waiting_extras_image → ждёт фото реквизита

StyleFlow:
  waiting_panties     → ждёт фото трусов
  choosing_style      → показано меню стилей
  choosing_extras     → показано меню доп. элементов
  waiting_extras_text
  waiting_extras_image

DescribeFlow:
  waiting_description → ждёт текстовое описание стиля
  waiting_panties     → ждёт фото трусов
  choosing_extras     → показано меню доп. элементов
  waiting_extras_text
  waiting_extras_image
```
